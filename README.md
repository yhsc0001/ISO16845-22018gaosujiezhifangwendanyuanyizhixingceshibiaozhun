## 标准内容概述

ISO 16845-2:2018标准规定了ISO 11898-2:2016中标准化的CAN物理层的一致性测试计划。该标准涵盖了静态和动态测试，其中动态测试包括ISO 11898-1:2015中标准化的部分实现的经典CAN协议和CAN FD协议的测试案例。静态测试则描述了需要在数据表中提供的数据。
